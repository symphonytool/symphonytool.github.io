Welcome to The Symphony IDE.

The Symphony IDE is an open-source tool originally developed in the EU COMPASS (Comprehensive Modelling for Advanced Systems of Systems) research project. The focus is on modelling systems of systems. The IDE is based on the Overture platform.

Getting the Symphony IDE.

Downloads of ready-to-run versions of the Symphony IDE are presently hosted on SourceForge; the current version is Release 0.2.0, and version 0.2.2 will be released by the end of November 2013.

Getting documentation.

Documentation links will be added with the 0.2.2 release of the Symphony IDE.

Getting the code.

All of the Symphony code is hosted on GitHub, under the @symphonytool organisation account. Everyone is welcome to grab it, fork it, and contribute.
